Stuart's dotfiles
=================


Installation
------------

1. Clone the repository

    ```bash
    $ git clone --recurse-submodules https://github.com/stuartcarnie/dotfiles ~/.dotfiles
    ```

2. Setup `stow`

    ```bash
    $ stow -t ~ stow
    ```
    
3. Install individual packages

    ```bash
    $ stow <directory>
    ```
